Move your container

Create a new image of your container sudo docker commit -m "ubuntu + tomcat" -a "Author SebLL" 16d905eaa267 myubu/tomcat:v1

Save this new image sudo docker save -o /tmp/myubutomcat.tar myubu/tomcat

Move your image to the SLES12 server cd /tmp; scp myubutomcat.tar root@10.3.57.168:/tmp

Connect to the SLES12 server ssh root@10.3.57.168 , go into /tmp to check the image is there

Check the existing docker images

Load the new repository docker load < myubutomcat.tar

Check the image is loaded docker images

Deploy a container from this image docker run -it -p 8889:8080 myubu/tomcat:v1 /bin/bash

Access the application

Start tomcat bin/startup.sh

Access Tomcat web page

Exiting the container without stopping it Use Ctrl+P+Q key combination

Copy a web site to the container docker cp /tmp/site.tar 16d905eaa267:/tmp where 16d905eaa267 is the ContainerID

Monitor the docker container docker stats ContainerID
