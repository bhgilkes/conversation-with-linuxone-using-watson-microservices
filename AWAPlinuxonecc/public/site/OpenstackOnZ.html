Using RESTClient for Openstack

Install firefox RESTClient search in the addons for restclient and install it ! Restart your firefox browser !

Launch RESTClient Use the icon on the firefox browser. You are now ready to perform REST API calls

Openstack API version Check the api versions available in your openstack cloud. Use Method GET and type in: http://10.3.57.159:5000/ . Switch to the tab "response Body Preview" to see the response

Authenticate first !

  • URL: http://10.3.57.159:5000/v3/auth/tokens
  • Method: POST
  • Request Headers: Content-type : application/json
  • Request Boby: Copy this content And adapt the username & password to your environment
  • How it looks like ?: View a Result Sample Yo ucan now perform any additional REST API call you like to the openstack cloud running on your Linux on z System or LinuxONE Systems

Validate the token use the previous value of "X-Subject-Token"

  • URL: http://10.3.57.159:5000/v3/auth/tokens
  • Method: GET
  • Request Headers: X-Auth-Token with the previous value from "X-Subject-Token"
  • Request Boby: Copy this content And adapt the username & password to your environment
